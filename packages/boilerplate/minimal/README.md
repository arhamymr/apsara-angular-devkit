# {{PROJECT_NAME}}

A minimal Angular + Tailwind boilerplate built with [apsara-ui](https://github.com/apsara/apsara-ui).

## Getting Started

```bash
npm install
npm start
```

## Adding Components

```bash
# Add button, input, and card components
npx apsara-ui add button input card

# List available components
npx apsara-ui list
```

## Project Structure

```
src/
├── app/
│   ├── ui/           # UI components (button, input, card, etc.)
│   ├── pages/        # Page components
│   ├── app.component.ts
│   ├── app.config.ts
│   └── app.routes.ts
├── styles.css
├── main.ts
└── index.html
```
